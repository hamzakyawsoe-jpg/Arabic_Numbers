الأعداد العربية

عدد (ကိန်းဂဏန်း) နှင့် معدود (ရေတွက်ခံပစ္စည်း) ကို ကျွမ်းကျင်လာစေရန် တဆင့်ချင်း လေ့လာကြရအောင်

🌙 ဘာကြောင့် ဒါက ခက်တာလဲ?

အာရဗီဘာသာတွင် ကိန်းဂဏန်းများသည် ရေတွက်ခံပစ္စည်း (معدود) ၏ ကျားမနှင့် ပမာဏပေါ် မူတည်၍ မတူညီသောပုံစံများ ရှိပါသည်။ ဤသည်မှာ အာရဗီဘာသာရပ်တွင် အနုပညာဆန်ဆုံးသော အပိုင်းတစ်ခုဖြစ်ပါသည်။

အဓိကကျသောသဘောတရားနှစ်ခုကိုသာ မှတ်ထားပါ — عدد (ကိန်းဂဏန်း) နှင့် معدود (ရေတွက်ခံပစ္စည်း)။ ဤနှစ်ခုကို ချိတ်ဆက်ပုံသည် အလွန်ရိုးရှင်းသော ယုတ္တိဗေဒပေါ်တွင် တည်ဆောက်ထားပါသည်။

معدود (ရေတွက်ခံပစ္စည်း)
🇲🇲 သုံးလုံးသောကုလားထိုင်များ
🗺️ ကိန်းဂဏန်းနည်းဥပဒေ — ခြုံမြင်ကွင်း
ကိန်းအပိုင်း ကျားမ معدود ပုံစံ အမှတ်အသား
١ — ၁ ကိုက်ညီသည် မဖrators مُفْرَد
٢ — ၂ ကိုက်ညီသည် ဒွိကိန်း مُثَنَّى
٣–١٠ ကိုလှည့်သည် အများကိန်း جَمْع
١١–١٢ ကိုက်ညီသည် မဖrators مُفْرَد
١٣–١٩ ကိုလှည့်သည် မဖrators مُفْرَد
٢٠, ٣٠... မပြောင်း မဖrators مُفْرَد
١٠٠, ١٠٠٠... မပြောင်း မဖrators مُفْرَد
အရေးကြီးဆုံးသောဥပဒေမှာ ၃–၁၀ ၏ "ကျားမကိုလှည့်ခြင်း" ဖြစ်သည်။ ဤသဘောတရားကို သဘောပေါက်သောအခါ ကျန်ကိန်းများ ယုတ္တိကျလာပါသည်။
1️⃣ ကိန်း ١ (တစ်) — واحد
📌 နည်းဥပဒေ
• ကိန်း "တစ်" သည် မရှိဘဲ ရေတွက်ၜ​ပေးနိုင်ပါသည် (မြန်မာဘာသာတွင် "တစ်လုံး" မထည့်ဘဲ "ကုလားထိုင်" ဟုပြောသလိုပင်)
• ရေတွက်ချင်ပါက "တစ်" သည် معدود ၏ ကျားမနှင့် ကိုက်ညီရပါသည်

ကျား (مذكر)

كِتَابٌ وَاحِدٌ
🇲🇲 စာအုပ်တစ်အုပ်
كِتَاب = ကျား

မ (مؤنث)

مَدْرَسَةٌ وَاحِدَةٌ
🇲🇲 ကျောင်းတစ်ကျောင်း
مَدْرَسَة = မ (ة ပါ)
تاء مربوطة (ة) ပါသောစကားလုံးများသည် မ (مؤنث) ဖြစ်ရလေ့ရှိသည်။ واحد (ကျား) → واحدة (မ)
2️⃣ ကိန်း ٢ (နှစ်) — اثنان / اثنتان
📌 နည်းဥပဒေ
• ကိန်း "နှစ်" သည် ဒွိကိန်း (مثنى) ကိုအသုံးပြုသည်
• "نِ-" သို့မဟုတ် "يِ-" ပြင်ဆင်ချက် (suffix) ဖြင့် ဖော်ပြသည်
• ကိန်းနံပါတ်ထည့်ရန် မလိုဘဲ ဒွိကိန်းပုံစံသာ သုံးနိုင်သည်

ကျား

كِتَابَانِ / كِتَابَيْنِ
🇲🇲 စာအုပ်နှစ်အုပ်

မ

مَدْرَسَتَانِ / مَدْرَسَتَيْنِ
🇲🇲 ကျောင်းနှစ်ကျောင်း
-انِ ကို nominative တွင်၊ -يْنِ ကို genitive/accusative တွင် သုံးသည်
🔑 ၃–၁၀ ၏ မာဂျစ် — ကျားမကိုလှည့်ခြင်း
⚡ အရေးကြီးဆုံးသောနည်းဥပဒေ
ကိန်း ၃ မှ ၁၀ အထိ — ကိန်းဂဏန်းသည် معدود ၏ ကျားမနှင့် ဆန့်ကျင်ဘက်ကျားမ ဖြစ်ရပါသည်!

معدود မဖြစ်က
ثَلَاثٌ
عدد ကျား-ပုံစံ (-ة မပါ)
မှတ်နည်း: "ة ပါသည်" ဆိုသည်မှာ ကိန်းဂဏန်းကျားပုံစံ (ة) သည် ကျားပစ္စည်းကို ရေတွက်သည်ဟု ထင်မြင်မိနိုင်သော်လည်း တကယ်တမ်းတွင် မပုံစံဖြစ်သည် — ၎င်းသည် ကျားပစ္စည်းကို ရေတွက်သည်!

🔟 ကိန်း ١١–١٢ (တစ်ဆယ့်တစ်, တစ်ဆယ့်နှစ်)
📌 နည်းဥပဒေ
• ကိန်းနှစ်ပိုင်းစလုံး (ones + tens) သည် معدود ၏ ကျားမနှင့် ကိုက်ညီရပါသည်
• معدود သည် مفرد (မူလပုံ) ဖြစ်ရပါသည် — منصوب (ا- accusative) ဖြင့်
